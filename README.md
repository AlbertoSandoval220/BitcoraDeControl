# Bitacora De Control
Control de Usuarios para el laboratorio de Compuaula
# 
Esta aplicación se encarga de controlar el acceso a los usuarios del laboratorio de cómputo del CETis No.44
El principal funcionamiento es bloquear las pantallas de los equipos que son utilizados principalmente por alumnos. Para poder ingresar o "desbloquear" las máquinas,
los alumnos tendrán que loggearse con su número de matricula, si este es correcto les brindará acceso a la máquina, no sin antes, pasar por un formulario el cual muestra
los datos del usuario, desde nombre, semestre, carrera, etc.
El alumno podrá seleccionar desde una lista desplegable al docente con el cuál se encuentra tomando clase y posteriormente, tendrá la posibilidad de llenar un campo de 
texto en donde agregará cualquier reporte de incidencias, anomalías o averías que este visualice en el equipo.
